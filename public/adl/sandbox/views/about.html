Virtual World Creator
The Virtual World Creator (VWC) brings interactive simulation authoring to the Web. Using the latest browser technologies, we let teachers and students create, edit and share interactive experiences without the need to send executable files across email, or to ask your students to download or install anything. Students can access and create worlds from any WebGL engabled device or browser!
Virtual Worlds in your Browser!
Students and teachers have access to hundreds of themed curriculum models and objects to build with. This opens up new door for immersive learning experiences and projects.

Virtual World Creator Platform
The Virtual World Creator is an immersive learning platform for K-12 learning. The VWC is an architecture for allowing students to build collaborative web applications - any application based on the framework allows naturally for multiple users to interact in real-time.
